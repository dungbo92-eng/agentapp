# User Profile

## 언어 / 응답

- 기본 응답 언어: 한국어
- 선호 스타일: 짧고 명확한 명령서형 설명, 표와 bullet 적극 활용
- 사용자는 여러 AI 개발 에이전트를 병렬/순차로 운용한다.

## 작업 선호

- 의사결정이 필요 없는 개발 작업은 에이전트가 능동적으로 이어가길 원한다.
- “다음 진행사항 진행”을 반복 입력하지 않아도 작업이 굴러가는 구조를 선호한다.
- 여러 에이전트가 같은 memory/plan을 공유하고 git으로 동기화되는 방식을 선호한다.

## 안전 선호

- 안전한 로컬 작업은 자동 진행을 원한다.
- 운영계 쓰기, 결제, 배포, 계정/요금/토큰 우회성 자동화는 명확한 보류/승인 큐로 분리한다.
